# NGO Fund and Donor Management System

## Overview
The NGO Fund and Donor Management System is designed to streamline the process of managing donations and donor information for non-profit organizations. It includes various features to enhance the operational efficiency of NGOs and improve the engagement with donors.

## Features
- **Donation Recording**: Track donations received from various sources and maintain a comprehensive record for financial reporting.
- **Donor Tracking**: Maintain detailed information about donors, including their contact details, donation history, and preferences.
- **Automated Thank-You Messages**: Automatically send thank-you messages to donors after receiving their contributions, enhancing donor relationships.
- **Financial Reports**: Generate financial reports to analyze the funding received and expenditures, providing insight into the NGO's financial health.

## Technical Stack
The frontend of the application is built using:
- **HTML**: For structuring the web pages.
- **CSS**: For styling and layout to create a visually appealing user interface.
- **JavaScript**: For implementing dynamic features and enhancing user experience.

## Conclusion
This system aims to provide a user-friendly interface for NGOs to manage their donations and donor relationships effectively, ensuring transparency and accountability.
