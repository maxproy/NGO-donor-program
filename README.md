# NGO Project Documentation

## Project Overview
This project aims to improve the effectiveness of non-governmental organizations (NGOs) by providing them with an integrated platform for managing their operations.

## Dual-Panel Architecture
The application follows a dual-panel architecture, separating contextual information and task execution to enhance usability and efficiency.

## Feature Details
1. **User Management**: Role-based access control and user authentication.
2. **Project Tracking**: Monitor project milestones and deliverables.
3. **Reporting Tools**: Generate comprehensive reports for stakeholders.
4. **Communication Channels**: Integrated messaging system for team collaboration.
5. **Data Analytics**: Insights into project performance through data visualization.

## Tech Stack
- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- Hosting: AWS

## Project Structure
```
/NGO-Project
├── client   # Frontend code
├── server   # Backend code
└── docs     # Documentation files
```

## Usage Guide
1. Clone the repository using `git clone <repo-url>`.
2. Install dependencies:
   - For the client: `cd client && npm install`
   - For the server: `cd server && npm install`
3. Start the development server:
   - For the client: `npm start`
   - For the server: `node server.js`

## Data Storage Information
Data is stored in a MongoDB database, which handles user data, project details, and logs.
